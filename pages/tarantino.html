a simple dress-up game created with HTML, CSS and JS where you can create an outfit using items from famous quentin tarantino movies.

INTRODUCTION

dress-up game coded with HTML, CSS and javascript. all illustrations were originally made for this project and this was my first experience with javascript.

PROCESS

For this project there weren't any guidelines as to what kind of product had to be made - as long as Javascript was used.

I decided to make a dress-up game with Tarantino characters. Pretty quickly I had an idea for the layout and I stuck with it throughout the project.

I had never worked with Javascript before so I struggled a lot with the coding, but I learned a lot. Writing code is truly trial and error - a single comma can break your entire website.

FOR THIS PROJECT I:

  • drew all of the illustrations
  • used functions, parameters, if/else statements
  • coded the game with html, css and javascript

END RESULT

This Dress-up Game has multiple categories: skin, hair, top, bottom, shoes, accessories & backgrounds.

Different kinds of outfits were directly inspired from the Quentin Tarantino movies Django Unchained, Inglorious Bastards and Pulp Fiction.

Of course, all combinations can be made. When the Kill Bill combination is completely selected, the layout will change to red/yellow and the Kill Bill siren will be heard.
