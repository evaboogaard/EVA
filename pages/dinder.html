INTRODUCTION

dinder is a matching application in which you can match with restaurants in amsterdam. users can fill in their preferences, set filters and swipe cards left and right. the project was created with handlebars, mongodb as a database and nodejs.

PROCESS

this project was the first experience i had with using templating engines, git, doing code-based projects with other people and streamlining how this works (with pull requests, etc)

i had to delve into some back-end code, which i didn't like, but it taught me a lot. i created all of the visual design myself and did all the css. aside from that, i created the login functionality. more on the process can be found in the repo's wiki.

END RESULT

me and my teammates created a matching application for restaurants in amsterdam with working login, filter and nodemailer functions. if you want to see the final project, you have to install it using the instructions in the readme.
