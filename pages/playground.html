INTRODUCTION

for this school assignment the intent was to learn more about CSS and get creative with it. the goal of the assignment was to pick a character or an artwork and recreate it with only div's and CSS.

PROCESS

starting this project off, i didn't have that much experience in CSS yet. i started making a breakdown sketch of the character i wanted to recreate: 'big hero 6' baymax. all of his different body parts were put into different divs and spans in my HTML document.

after the HTML was written, I started styling the different parts. I used a lot of different shadows, border-radiuses and z-index values to create the character. this challenge has taught me a lot about positioning, variables and the complexity of styling.

END RESULT

i created baymax and a custom gradient-heavy background using CSS only. i added in some animations to make him, the sun, moon and stars move. using a JS toggle function that swaps the :root color variables you can also switch to dark mode.
